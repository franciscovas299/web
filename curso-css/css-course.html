Curso de CSS

Sección 1: Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje que usamos para estilizar una página HTML.

Ejemplo de CSS:

body {
  font-family: Arial, sans-serif;
}
h1 {
  color: blue;
}
p {
  font-size: 14px;
}
