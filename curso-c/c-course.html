Fundamentos de C

Sección 1: Introducción a C

C es un lenguaje de programación de propósito general que se utiliza ampliamente en el desarrollo de sistemas y aplicaciones.

Ejemplo de C:

#include <stdio.h>
int main() {
    printf("Hola, Mundo!\n");
    return 0;
}
