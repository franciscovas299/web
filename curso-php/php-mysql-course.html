Sección 3: Conexión a MySQL con PHP

Aprende a conectar PHP con una base de datos MySQL para realizar operaciones CRUD.

Ejemplo de Conexión:

<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "miDB";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}
echo "Conexión exitosa";
?>

Sección 4: Operaciones CRUD

Las operaciones CRUD permiten crear, leer, actualizar y eliminar registros en una base de datos.

Ejemplo de Inserción:

<?php
$sql = "INSERT INTO usuarios (nombre, email) VALUES ('Juan', 'juan@example.com')";
if ($conn->query($sql) === TRUE) {
    echo "Nuevo registro creado exitosamente";
} else {
    echo "Error: " . $sql . "
" . $conn->error;
}
$conn->close();
?>

2. ¿Cómo insertas un nuevo registro en la tabla "usuarios"?
