Sección 1: Funciones Avanzadas

En esta sección, aprenderás sobre las funciones avanzadas en JavaScript, como las funciones de flecha y las funciones de orden superior.

Ejemplo de funciones de flecha:

const suma = (a, b) => a + b;
console.log(suma(2, 3)); // 5

Preguntas

1. ¿Cómo se define una función de flecha en JavaScript?




2. ¿Qué retorna la siguiente función de flecha: const cuadrado = n => n * n; cuando se llama con cuadrado(4)?




Sección 2: Promesas y Async/Await

En esta sección, aprenderás sobre las promesas y la sintaxis de async/await para manejar operaciones asíncronas en JavaScript.

Ejemplo de Promesas:

const promesa = new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve('¡Éxito!');
  }, 1000);
});

promesa.then(mensaje => {
  console.log(mensaje);
});

Preguntas

1. ¿Qué hace el método .then() en una promesa?




2. ¿Qué palabra clave se usa para esperar la resolución de una promesa dentro de una función asíncrona?

Sección 5: Clases y Módulos

En esta sección, aprenderás sobre las clases y los módulos en JavaScript.

Ejemplo de clase:

class Persona {
  constructor(nombre, edad) {
    this.nombre = nombre;
    this.edad = edad;
  }

  saludar() {
    console.log(`Hola, mi nombre es ${this.nombre}`);
  }
}

const juan = new Persona('Juan', 30);
juan.saludar(); // Hola, mi nombre es Juan

1. ¿Cómo se define una clase en JavaScript?




2. ¿Qué palabra clave se usa para exportar un módulo en JavaScript?
