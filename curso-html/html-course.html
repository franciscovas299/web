Curso de HTML

Sección 1: Introducción a HTML

HTML (HyperText Markup Language) es el estándar para crear páginas web. Aquí aprenderás los conceptos básicos.

Ejemplo de HTML:

<!DOCTYPE html>
<html>
  <head>
    <title>Página de ejemplo</title>
  </head>
  <body>
    <h1>Hola, Mundo!</h1>
    <p>Este es un párrafo de ejemplo.</p>
  </body>
</html>
